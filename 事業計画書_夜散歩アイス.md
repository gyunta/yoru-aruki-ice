# 事業計画書

**事業名：夜散歩アイス（仮称）**
作成日：2026年5月31日　（第2版）

---

## 1. 事業コンセプト

| 項目 | 内容 |
|------|------|
| コンセプト | 多摩川沿いの夜散歩を楽しむ人に、本格ジェラートと会話を届ける二人組の超小規模個人事業 |
| キャッチフレーズ（案） | 「川沿いの夜風と、とろけるジェラート」 |
| 事業形態 | 個人事業（趣味の延長・副業スタート） |
| 運営者 | カップル2名（武蔵新田在住） |
| 強み | 二人のコミュニケーション力 × IT・SNS発信スキル × 地元密着 |

---

## 2. 事業の背景・動機

- 多摩川沿いは夕方〜夜にかけて散歩・ランニング・犬の散歩をする人が多く、飲食の自然な需要がある
- 夜間に本格アイスを提供するPOPUPは極めて少なく、希少性・体験価値が高い
- カップル二人で運営することで接客・オペレーションを分担し、余裕ある運営と高い顧客体験を両立
- 趣味の延長として始めることで、無理なく継続できる持続可能な形を目指す
- 利益最大化よりも「赤字にならない・楽しく続ける」を最優先とする

## 5. 役割分担

二人の強みを活かして、現場と発信を同時並行で回す体制を構築する。

| 役割 | 主担当 | 内容 |
|------|--------|------|
| 接客・販売 | 二人で交互・並行 | お客さんへの声かけ、注文受付、ジェラート提供 |
| オペレーション | 主に一方 | 在庫管理・温度管理・補充・会計 |
| SNSリアルタイム発信 | 主に一方 | 営業中の写真・動画撮影、Instagram/Xへの即時投稿 |
| Web・IT管理 | 二人で分担 | サイト更新、出店情報管理、データ分析 |
| 仕入れ・準備 | 二人で分担 | 事前の仕入れ、備品準備、出店場所の確保 |

> **ポイント：** 一人が接客に集中できるよう、もう一人がSNS撮影・発信を担当。コミュニケーション力の高さを最大限に活かし、「また会いたい」と思われる接客体験を目指す。

## 6. IT・SNS戦略

### 6-1. Webサイト（専用サイト構築）

Claudeを活用して本格的なWebサイトを自作し、ブランドの核として機能させる。

| 項目 | 内容 |
|------|------|
| 目的 | 出店情報のハブ・ブランドの顔・SNSへの流入先 |
| 構成ページ | トップ／出店スケジュール／メニュー・フレーバー紹介／ギャラリー／about（二人のストーリー） |
| 技術スタック | Claude活用でゼロから制作（HTML/CSS or 静的サイトジェネレーター） |
| ドメイン | 独自ドメインを取得し、信頼感・ブランド感を演出 |
| 更新頻度 | 出店のたびにスケジュール・メニューをリアルタイム更新 |

### 6-2. SNS戦略

| プラットフォーム | 役割 | 発信内容 |
|----------------|------|---------|
| **Instagram** | メイン集客・ブランディング | ジェラートの美しい写真・川沿いの夜景・二人の雰囲気・リール動画 |
| **X（旧Twitter）** | リアルタイム告知 | 当日出店情報・天候による中止・フレーバー速報 |
| **Instagram Stories** | 当日告知・舞台裏 | 「今日出ます！」「残りわずか」などライブ感のある投稿 |

### 6-3. コンテンツ戦略（差別化ポイント）

- **ストーリーで売る：** 商品写真だけでなく「二人で準備する様子」「多摩川の夜景」「お客さんとの会話」を発信し、ブランドへの共感を醸成
- **出店前後のルーティン投稿：** 「今週末出ます」→「出店中」→「ありがとうございました」の流れをルーティン化して期待感を作る
- **フレーバー紹介コンテンツ：** 季節のフレーバーを毎回丁寧に紹介し、「今週は何が出るか」を楽しみにしてもらう
- **ハッシュタグ設計：** `#多摩川` `#武蔵新田` `#ジェラート` `#夜散歩` などローカル×カテゴリで検索流入を狙う

## 9. リスクと対策

| リスク | 対策 |
|--------|------|
| 雨天・悪天候による集客不足 | 天気予報に応じて柔軟にキャンセル。POPUPなので固定費ゼロでリスク最小化。SNSで早めに中止告知 |
| アイスの温度管理 | 保冷ボックスとドライアイスで管理。売れ残りは持ち帰り消費 |
| 許認可の取得遅れ | 早めに地元保健所へ相談。先行して資格取得を進める |
| SNS発信が続かない | 二人で発信役を分担し、どちらかが投稿できる体制に。テンプレ化でハードルを下げる |
| 競合の出現 | 二人の接客・体験価値・ブランドストーリーで差別化。代替不可能な「人」を軸にする |

## 11. 事業の方針・大切にすること

- **黒字維持を最低ライン**とし、無理な拡大はしない
- 趣味・楽しさ・二人の時間を失わないペースで継続する
- お客さんとの会話・交流を大切にし、「川沿いの顔なじみ」になる
- SNSは義務感でなく「楽しんで発信する」ことを意識する
- 将来的な方向性（継続・拡大・撤退）は1年後に改めて二人で判断する
