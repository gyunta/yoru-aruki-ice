# 事業計画書　夜散歩アイス

*作成日：2026年5月31日　版：v1.0（仮置き数字）*

---

## 1. 事業コンセプト

> **「夜の多摩川で、立ち止まるひとときを。」**

多摩川沿い（武蔵新田周辺）を夜散歩する人々に向けて、本格ジェラートをPOPUP販売する小規模事業。
18〜21時という「夕涼みタイム」に、川沿いの景色と一緒に楽しめるアイスの体験を提供する。

---

## 2. 事業の背景・動機

- 多摩川沿いは夜でも散歩・ジョギング・ピクニックをする人が多く、既存の飲食店が少ない時間帯がある
- カップル2名の共同プロジェクトとして、楽しみながら収益を出せる形を模索
- 超小規模スタートで赤字を出さないことを最優先とし、手応えがあれば拡大を検討

## 6. IT・SNS戦略

### 発信チャネル
- **Instagram**：商品写真・川沿いの雰囲気・出店告知（ビジュアル重視）
- **X（旧Twitter）**：当日の出店情報・リアルタイム告知・軽いコンテンツ
- **Webサイト**：出店スケジュール・メニュー・PLシミュレーター公開

### Webサイト構成（予定）
```
index.html       ← トップ・出店スケジュール
simulator.html   ← PLシミュレーター（公開版）
style.css        ← 共通スタイル
```

### 運用方針
- Claude Codeを活用してコーディングレスで更新可能な体制を構築
- GitHub Pagesで無料ホスティング（独自ドメインは将来検討）

## 9. リスクと対策

| リスク | 対策 |
|--------|------|
| 天候不順による出店中止 | SNSで事前告知・振替日を設定。在庫を最小限に仕入れ |
| 販売数が想定を下回る | 仕入れ数を絞り、売り切りで運営。在庫ロスを防ぐ |
| 出店場所の許可取得 | 公共スペースの利用ルールを事前確認・遵守 |
| 食品衛生リスク | 食品衛生責任者資格取得を検討。保冷管理を徹底 |
| モチベーション低下 | 楽しさ優先・無理のないペース設定。赤字なら即縮小 |
